# Deus

Deus is a lightweight, local voice assistant written in Python. It is a limited-feature Siri-style clone focused on basic voice input, command recognition, and simple responses.

The project is experimental and intentionally minimal.

## What it does

- Listens for voice input
- Converts speech to text
- Matches spoken commands
- Executes simple predefined actions or responses

## Requirements

- Python 3.10 or newer
- A working microphone
- Required Python libraries listed in `requirements.txt`

## Usage

Run the assistant:

```bash
python main.py
